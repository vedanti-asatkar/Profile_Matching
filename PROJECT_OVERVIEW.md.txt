## Project Overview
This project is an AI-powered profile-based matching platform that recommends professionally compatible users based on:
- NLP semantic similarity
- MBTI work-style compatibility
- location matching
- adaptive feedback learning

The system is deployed using Streamlit Cloud and supports personalized recommendations for each user profile.
